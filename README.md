# Govt. MC High School Website

This is a frontend school website project created completely from scratch using **HTML** and **CSS**. It details the web presence of "Govt. MC High School" (Bright Future School), providing a clean and visually appealing interface to deliver information to students and parents.

## 🚀 Features

- **Home Page** (`index.html`): Includes a welcoming hero section, school features, statistics (students, teachers, etc.), an image gallery, and a footer with contact details.
- **About Page** (`About.html`): Provides deeper insight into the school's mission, values, and environment.
- **Admissions Page** (`Admission.html`): Contains necessary information regarding the school's admission process.
- **Custom Styling**: Hand-written, vanilla CSS (`style.css`, `About.css`, `Admission.css`) structured for readability and maintainability.

## 📁 Project Structure

```text
project2/
│
├── index.html       # Main landing page
├── About.html       # About section page
├── Admission.html   # Admissions page
│
├── style.css        # Styles tailored for the home page
├── About.css        # Styles tailored for the about page
├── Admission.css    # Styles tailored for the admissions page
│
└── img*.jpg         # Various images used for the gallery and hero sections
```

## 🛠️ Technologies Used

- **HTML5**: For semantic structuring and building the content of the web pages.
- **CSS3**: For all visual styling, layout management, and user interface enhancements.

## 💻 How to Run

1. Clone or download this project's files onto your local machine.
2. Navigate to the `project2` folder.
3. Simply double-click `index.html` to open it in your default web browser. 
*(Alternatively, you can use an extension like VS Code "Live Server" for auto-reloading during development).*
